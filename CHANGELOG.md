## [0.1.1] - 2026-06-20

### Added
- Додано централізоване логування застосунку (console + file logging).
- Додано middleware для логування всіх HTTP запитів (метод, шлях, статус, час виконання, IP).
- Додано `lifespan` для управління життєвим циклом FastAPI застосунку.
- Додано бібліотеку `pydantic-settings`
- Додано файл .env.example з прикладом конфігурації застосунку.
- Додано підтримку конфігурації через файл .env.
- Додано централізовану конфігурацію застосунку через settings.py.
- Додано налаштування параметрів бази даних через Pydantic Settings.
- Додано конфігурацію `Dockerfile` для контейнеризації FastAPI застосунку на базі легкого образу Python Slim.
- Додано файл `docker-compose.yml` для мультиконтейнерного розгортання застосунку та бази даних PostgreSQL 16.
- Додано автоматичне монтування локальної директорії (`volumes`) та режим `--reload` для зручної локальної розробки в Docker.
- Інтегровано та повністю налаштовано Alembic для асинхронних міграцій через `asyncpg`.
- Згенеровано першу базову міграцію для ініціалізації структури бази даних.
- Додано автоматичну перевірку з'єднання з базою даних (healthcheck) при старті через `check_db_connection`.
- Створено базовий шаблон `base.html` на Jinja2 для уникнення дублювання HTML коду.

### Changed
- Оновлено ініціалізацію логера для підтримки uvicorn логів.
- Додано в `lifespan` логування базової директорії та url бази даних
- Оновлено команду запуску Uvicorn в Docker-контейнері для роботи з кореневим файлом `main:app`.

### Fixed
- Покращено стабільність логування при reload режимі uvicorn.

## [0.2.0] - 2026-06-20

### Added
- Додано властивості `created_at_local` та `updated_at_local` для автоматичної конвертації часу сервера з UTC у Київський часовий пояс при відображенні.
- Додано SQLAlchemy модель `User` з полями для email, хешованого пароля, дати народження та статусу активності.
- Згенеровано та застосовано асинхронну міграцію Alembic для створення таблиці `users` у базі даних PostgreSQL.
